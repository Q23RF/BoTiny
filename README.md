## 指令列表
### 說明
- **!cmd**: 查看指令列表
### 推薦
- **!fancam**: 推薦梯子直拍
- **!stage**: 推薦梯子舞台
- **!song**: 推薦梯子歌曲
### 互動
- **!poll+問題+選項(最多10項)**: 在當前頻道開啟投票
- **!draw**: 隨機抽取並標記伺服器成員
- **!submit+投稿內容**: 在#廁所頻道發布匿名投稿內容
